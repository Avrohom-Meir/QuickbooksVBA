Welcome to QuickBooks VBA Integration

This platform provides seamless integration between your Microsoft Access applications and QuickBooks Online using secure and supported API methods.

Developed by PROPEL SOFTWARE LIMITED, this solution helps automate finance operations and simplifies synchronisation between desktop databases and cloud-based accounting tools.

Key Features

  • Real-time syncing of invoices, customers, and payments from Access to QuickBooks.
  • Secure OAuth 2.0 connection and token refresh management.
  • Supports multi-company workflows using Realm ID mapping.
  • Advanced error logging and recovery handling.
  • GDPR-compliant architecture with robust data protection practices.
  • Sandbox environment available for safe development and testing.
